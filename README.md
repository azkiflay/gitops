# Introduction
GitHub Actions is an end-to-end workflow automation platform that handles continuous integration and continuous delivery (CI/CD) aspect for Git and GitHub. As a CI/CD automation tools, GitHub Actions helps deliver software as soon as possible to its users. While GitHub allows collaboration on writing software code, GitHub Actions enables fast delivery of the software to users. Prior to GitHub Actions, other tooling (e.g., Jenkins, Travis CI) needed to be integrated on GitHub to facilitate CI/CD.
- GitHub Actions is integrated to other GitHub components.

While GitHub Actions is one of the popular tools, there are other alternatives such [Azure DevOps](https://azure.microsoft.com/en-us/products/devops), [Jenkins](https://www.jenkins.io/), [Travis CI](https://www.travis-ci.com/), and [CircleCI](https://circleci.com/).
